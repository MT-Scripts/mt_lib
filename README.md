# MT Lib (WIP)
FiveM lib with several UI elements and useful functions

# Features done:
- NPC Dialogue UI
- Mission status UI
- Key text UI
- Timer UI
- Interaction points
- Create ped
- Create blip
- Create prop

# Features coming:
- Notifications
- Some minigames
- Some more simple useful functions
- Sales stash for ox_inventory
- Items locked stash for ox_inventory
- And more...

# Dependicies
- ox_lib
